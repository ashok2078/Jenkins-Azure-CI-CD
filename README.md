# Jenkins CI/CD Pipeline on Microsoft Azure

## Project Overview

This project demonstrates how to install and configure Jenkins on an Azure Ubuntu 24.04 Virtual Machine and deploy a simple website to an Apache Web Server using a Jenkins Pipeline.

## Technologies Used

- Microsoft Azure
- Ubuntu 24.04 LTS
- Jenkins
- Apache2
- Git
- GitHub

## Project Architecture

GitHub Repository
        
        
Jenkins Pipeline
        
        
Apache Web Server
        
        
Website

## Project Files

- README.md
- NOTES.md
- Jenkinsfile
- index.html

## Jenkins Pipeline Stages

1. Build
2. Deploy
3. Verify

## Verification

After successful pipeline execution, open:

http://YOUR_PUBLIC_IP

The website should display the Jenkins CI/CD project page.
